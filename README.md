# Ясный

Язык программирования с русскоязычным синтаксисом. Версия 0.5.

```
конст имя = "мир"
вывод("Привет, {имя}!")
```

## Зачем нужен Ясный

Когда новичок без знания английского начинает учить программирование,
его внимание делится между алгоритмом и переводом ключевых слов. Что
такое `def`, `return`, `for in`, как читается `if-elif-else` — всё
это лишний слой между человеком и сутью того, что он пишет.

Ясный убирает этот слой. На нём можно изучить переменные, условия,
циклы, функции, классы — на родном языке. После этого переход на
Python или JavaScript занимает дни, а не месяцы: концепции уже
понятны, остаётся только запомнить английские слова.

Ясный не заменяет Python или JS. Это первый шаг, мост.

## Установка

Понадобится Go 1.22 или новее.

```
git clone https://github.com/warcorprp-web/yasny-lang.git
cd yasny-lang
go build -o yasny .
```

После сборки положите бинарник `yasny` в `$PATH`.

## Запуск

```
yasny файл.ya          # выполнить файл
yasny                  # интерактивный режим (REPL)
```

## Первая программа

Файл `привет.ya`:

```
вывод("Привет, мир!")
```

Запуск:

```
yasny привет.ya
```

## Основы языка

### Переменные

```
конст имя = "Иван"          # неизменяемая
перем счётчик = 0           # изменяемая
счётчик = счётчик + 1
счётчик += 5
```

`конст` — основная форма. `перем` используется только тогда, когда
переменную действительно нужно менять.

### Типы данных

```
конст число = 42
конст дробь = 3.14
конст текст = "строка"
конст флаг = да              # или нет
конст список = [1, 2, 3]
конст словарь = {"ключ": "значение"}
конст пусто = пусто
```

### Интерполяция строк

В двойных кавычках можно вставлять выражения в `{...}`:

```
конст имя = "Аня"
вывод("Привет, {имя}!")
вывод("2 + 2 = {2 + 2}")
```

В одинарных кавычках интерполяция не работает — это обычная строка.

### Условия

```
если возраст >= 18
    вывод("Взрослый")
иначеесли возраст >= 14
    вывод("Подросток")
иначе
    вывод("Ребёнок")
конец
```

Короткая форма для одного выражения:

```
если возраст >= 18: вывод("Взрослый") иначе: вывод("Ребёнок")
```

### Циклы

Со счётчиком:

```
для i от 1 до 10
    вывод(i)
конец

для i от 0 до 100 по 5    # с шагом 5
    вывод(i)
конец
```

По коллекции:

```
для имя в ["Аня", "Боря", "Вася"]
    вывод(имя)
конец
```

С условием:

```
перем k = 0
пока k < 5
    вывод(k)
    k += 1
конец
```

### Функции

```
функция приветствовать(имя)
    вернуть "Привет, {имя}!"
конец

вывод(приветствовать("Маша"))
```

Параметр по умолчанию:

```
функция степень(основание, показатель = 2)
    вернуть основание * основание
конец
```

Короткая форма с автоматическим возвратом значения:

```
функция квадрат(x): x * x
```

Лямбда:

```
конст удвоить = x => x * 2
```

### Классы

```
класс Точка
    функция создать(x, y)
        это.x = x
        это.y = y
    конец

    функция расстояние_до(другая)
        конст dx = это.x - другая.x
        конст dy = это.y - другая.y
        вернуть корень(dx * dx + dy * dy)
    конец
конец

конст a = Точка(0, 0)
конст b = Точка(3, 4)
вывод(a.расстояние_до(b))
```

Наследование:

```
класс Животное
    функция создать(имя)
        это.имя = имя
    конец

    функция говорить()
        вернуть "{это.имя} издаёт звук"
    конец
конец

класс Кот наследует Животное
    функция говорить()
        вернуть "{это.имя} мяукает"
    конец
конец

конст барсик = Кот("Барсик")
вывод(барсик.говорить())
```

`создать` — имя конструктора. Если в потомке его не объявить, будет
использован конструктор родителя.

### Сопоставление с образцом

```
конст имя = совпадение день
    когда 1: "Понедельник"
    когда 2: "Вторник"
    когда 3: "Среда"
    иначе: "Ещё какой-то день"
конец
```

### Обработка ошибок

```
попытка
    рискованная_операция()
поймать ошибка
    вывод("Не получилось: {ошибка}")
всегда
    очистить()
конец
```

Бросить ошибку:

```
если делитель == 0
    бросить "деление на ноль"
конец
```

### Тесты

Тесты — часть языка, отдельные библиотеки не нужны:

```
тест "сложение работает"
    проверить плюс(2, 3) == 5
    проверить плюс(-1, 1) == 0
конец
```

Запустите файл, чтобы увидеть результат каждого теста.

## Стандартная библиотека

### Вывод и ввод

- `вывод(значение)` — вывести в консоль
- `ввод(приглашение)` — прочитать ввод пользователя

### Преобразования

- `тип(x)` — тип значения
- `строка(x)`, `число(x)` — преобразование

### Числа и строки

- `длина(x)` — длина массива или строки
- `мин(a, b)`, `макс(a, b)`, `абс(x)`, `округл(x)`, `корень(x)`,
  `степень(основание, показатель)`

### Массивы

- `добавить`, `удалить`, `вставить`
- `фильтр`, `преобразовать`, `свернуть`, `найти`, `найти_индекс`
- `сортировать`, `реверс`, `объединить`, `соединить`
- `сумма`, `все`, `любой`, `взять`, `пропустить`
- `уникальные`, `объединение`, `пересечение`, `разность`

### Словари

- `ключи(объект)`, `значения(объект)`, `получить(объект, ключ, по_умолчанию)`

### Строки

- `верхний`, `нижний`, `обрезать`, `повторить`
- `разделить`, `соединить`, `заменить`, `подстрока`
- `содержит`, `начинается_с`, `заканчивается_на`

### Файлы

- `читать_файл(путь)`, `записать_файл(путь, содержимое)`,
  `существует_файл(путь)`

### Модули, доступные без импорта

- `мат` — числа и тригонометрия: `мат.пи`, `мат.е`, `мат.sin`,
  `мат.cos`, `мат.лог`, `мат.пол`, `мат.потолок`
- `время` — дата и время: `время.сейчас()`, `время.строка()`,
  `время.год()`, `время.спать(мс)`
- `json` — `json.создать(объект)`, `json.разобрать(строка)`
- `регвыр` — регулярные выражения: `регвыр.найти(текст, шаблон)`,
  `регвыр.заменить(текст, шаблон, замена)`,
  `регвыр.совпадает(текст, шаблон)`
- `http` — `http.получить(url)`, `http.пост(url, тело)`,
  `http.сервер(порт, обработчик)`
- `файлы`, `путь`, `ос`, `крипто`, `csv`, `случайное`, `cli`

## Модули

Файл `математика.ya`:

```
экспорт конст пи = 3.14159

экспорт функция квадрат(x)
    вернуть x * x
конец
```

В другом файле:

```

вывод(м.пи)
вывод(м.квадрат(5))
```

## Переход на Python и JavaScript

Все концепции переносятся один в один — меняются только ключевые
слова.

| Ясный            | Python              | JavaScript          |
|------------------|---------------------|---------------------|
| `конст x = 5`    | `x = 5`             | `const x = 5`       |
| `перем x = 5`    | `x = 5`             | `let x = 5`         |
| `функция f()`    | `def f():`          | `function f() {}`   |
| `вернуть x`      | `return x`          | `return x;`         |
| `если ... иначе` | `if ... else`       | `if ... else`       |
| `пока`           | `while`             | `while`             |
| `для x в y`      | `for x in y`        | `for (x of y)`      |
| `класс`          | `class`             | `class`             |
| `создать`        | `__init__`          | `constructor`       |
| `это`            | `self`              | `this`              |
| `да` / `нет`     | `True` / `False`    | `true` / `false`    |
| `пусто`          | `None`              | `null`              |
| `выдать`         | `yield`             | `yield`             |
| `асинх` / `ждать`| `async` / `await`   | `async` / `await`   |
| `конец`          | (отступы)           | `}`                 |

После этой таблицы и недели практики на Python/JS вы пишете на новом
языке без затруднений.

## Принципы языка

См. [PRINCIPLES.md](PRINCIPLES.md). Главное:

- Программа читается как алгоритм на русском.
- Слова важнее символов.
- Один способ сделать одно дело.
- Иммутабельность по умолчанию.
- Ошибки с подсказками.
