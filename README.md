# Ясный

Ясный — экспериментальный язык программирования на русском языке. Создан как исследовательский проект для изучения возможностей разработки на родном языке.

## О проекте

Это учебный проект, который показывает, как может выглядеть современный язык программирования с русским синтаксисом. Язык включает базовые конструкции и несколько интересных возможностей из современных языков.

Проект находится в активной разработке. Код может содержать ошибки, API может меняться. Не рекомендуется для production использования.

## Установка

### Из исходников

Требуется Go 1.18 или новее.

```bash
git clone https://github.com/warcorprp-web/prostoy-lang.git
cd prostoy-lang
go build -o yasniy main.go
```

Для установки в систему:

```bash
sudo cp yasniy /usr/local/bin/
```

### macOS

Готовый бинарник для Intel Mac:

```bash
curl -L https://github.com/warcorprp-web/prostoy-lang/raw/main/prostoy-0.42-macos-intel.dmg.tar.gz -o yasniy.tar.gz
tar -xzf yasniy.tar.gz
./install.command
```

## Первая программа

Создайте файл `hello.ya`:

```
вывод("Привет, мир!")
```

Запустите:

```bash
yasniy hello.ya
```

Или используйте интерактивный режим (REPL):

```bash
yasniy
>>> вывод("Привет!")
Привет!
>>> выход
```

## Основные возможности

### Переменные

```
конст имя = "Иван"           # Неизменяемая переменная
перем возраст = 25           # Изменяемая переменная
возраст = 26                 # Можно изменить
```

### Типы данных

```
конст число = 42             # Целое число
конст дробь = 3.14           # Дробное число
конст текст = "строка"       # Строка
конст флаг = да              # Булево значение (да/нет)
конст список = [1, 2, 3]     # Массив
конст данные = {"ключ": "значение"}  # Объект
```

### Интерполяция строк

```
конст имя = "Мария"
вывод("Привет, {имя}!")      # Привет, Мария!

конст x = 5
конст y = 10
вывод("Сумма: {x + y}")      # Сумма: 15
```

### Функции

```
функция сумма(a, b)
    возврат a + b
конец

конст результат = сумма(5, 3)
вывод(результат)             # 8
```

Лямбда-функции:

```
конст удвоить = х => х * 2
вывод(удвоить(5))            # 10
```

### Условия

```
конст возраст = 18

если возраст >= 18
    вывод("Совершеннолетний")
иначеесли возраст >= 14
    вывод("Подросток")
иначе
    вывод("Ребенок")
конец
```

### Циклы

Цикл по массиву:

```
конст числа = [1, 2, 3, 4, 5]

для число в числа
    вывод(число)
конец
```

Цикл с условием:

```
перем счетчик = 0

пока счетчик < 5
    вывод(счетчик)
    счетчик = счетчик + 1
конец
```

Цикл с диапазоном:

```
для i от 1 до 10
    вывод(i)
конец
```

### Классы

```
класс Человек
    функция конструктор(имя, возраст)
        это.имя = имя
        это.возраст = возраст
    конец
    
    функция представиться()
        вывод("Меня зовут {это.имя}, мне {это.возраст} лет")
    конец
конец

конст человек = новый Человек("Иван", 25)
человек.представиться()
```

## Примеры кода

### Работа с массивами

```
конст числа = [1, 2, 3, 4, 5]

# Фильтрация
конст четные = фильтр(числа, х => х % 2 == 0)
вывод(четные)                # [2, 4]

# Преобразование
конст удвоенные = преобразовать(числа, х => х * 2)
вывод(удвоенные)             # [2, 4, 6, 8, 10]

# Поиск
конст найденное = найти(числа, х => х > 3)
вывод(найденное)             # 4

# Сумма
вывод(сумма(числа))          # 15
```

### Деструктуризация

```
конст [первый, второй, ...остальные] = [1, 2, 3, 4, 5]
вывод(первый)                # 1
вывод(второй)                # 2
вывод(остальные)             # [3, 4, 5]
```

### Spread оператор

```
конст массив1 = [1, 2, 3]
конст массив2 = [4, 5, 6]
конст объединенный = [...массив1, ...массив2]
вывод(объединенный)          # [1, 2, 3, 4, 5, 6]
```

### Optional chaining

```
конст данные = {"пользователь": {"имя": "Иван"}}

вывод(данные?.пользователь?.имя)     # Иван
вывод(данные?.адрес?.город)          # нет (не ошибка)
```

### Pattern matching

```
конст день = 3

конст название = совпадение день
    когда 1: "Понедельник"
    когда 2: "Вторник"
    когда 3: "Среда"
    когда 4: "Четверг"
    когда 5: "Пятница"
    иначе: "Выходной"
конец

вывод(название)              # Среда
```

### List comprehension

```
конст числа = [1, 2, 3, 4, 5]
конст квадраты = [x * x для x в числа если x % 2 == 0]
вывод(квадраты)              # [4, 16]
```

### Работа с файлами

```
# Запись в файл
записать_файл("test.txt", "Привет, мир!")

# Чтение из файла
конст содержимое = читать_файл("test.txt")
вывод(содержимое)

# Проверка существования
если существует_файл("test.txt")
    вывод("Файл существует")
конец
```

### HTTP сервер

```
функция обработчик(запрос)
    возврат {
        "статус": 200,
        "тело": "Привет из Ясного!"
    }
конец

http_сервер(8080, обработчик)
вывод("Сервер запущен на порту 8080")
```

### JSON

```
конст данные = {"имя": "Иван", "возраст": 25}

# Преобразовать в JSON
конст json = json_создать(данные)
вывод(json)                  # {"имя":"Иван","возраст":25}

# Разобрать JSON
конст объект = json_разобрать(json)
вывод(объект.имя)            # Иван
```

## Встроенные функции

### Ввод/вывод

- `вывод(значение)` — вывести в консоль
- `ввод(приглашение)` — получить ввод пользователя

### Работа с типами

- `тип(значение)` — получить тип значения
- `строка(значение)` — преобразовать в строку
- `число(строка)` — преобразовать в число
- `длина(массив)` — получить длину массива или строки

### Строки

- `разделить(строка, разделитель)` — разделить строку
- `соединить(массив, разделитель)` — соединить массив в строку
- `заменить(строка, старое, новое)` — заменить подстроку
- `верхний(строка)` — в верхний регистр
- `нижний(строка)` — в нижний регистр
- `подстрока(строка, начало, конец)` — получить подстроку
- `обрезать(строка)` — удалить пробелы с краев
- `начинается_с(строка, префикс)` — проверить начало
- `заканчивается_на(строка, суффикс)` — проверить конец
- `содержит(строка, подстрока)` — проверить наличие
- `повторить(строка, количество)` — повторить N раз

### Массивы

- `добавить(массив, элемент)` — добавить элемент
- `удалить(массив, индекс)` — удалить элемент
- `вставить(массив, индекс, элемент)` — вставить элемент
- `найти(массив, функция)` — найти элемент по условию
- `найти_индекс(массив, функция)` — найти индекс элемента
- `фильтр(массив, функция)` — отфильтровать массив
- `преобразовать(массив, функция)` — преобразовать элементы
- `дляКаждого(массив, функция)` — выполнить для каждого
- `свернуть(массив, начальное, функция)` — свернуть в значение
- `все(массив, функция)` — проверить все элементы
- `любой(массив, функция)` — проверить хотя бы один
- `сортировать(массив)` — отсортировать
- `реверс(массив)` — развернуть
- `объединить(массив1, массив2)` — объединить массивы
- `взять(массив, количество)` — взять N элементов
- `пропустить(массив, количество)` — пропустить N элементов
- `сумма(массив)` — сумма элементов
- `установить(массив)` — уникальные элементы

### Математика

- `мин(a, b)` — минимум
- `макс(a, b)` — максимум
- `степень(основание, показатель)` — возведение в степень
- `корень(число)` — квадратный корень
- `абс(число)` — абсолютное значение
- `округл(число)` — округление
- `случайное()` — случайное число от 0 до 1
- `диапазон(начало, конец)` — создать диапазон чисел

### Объекты

- `ключи(объект)` — получить ключи
- `значения(объект)` — получить значения

### Файлы

- `читать_файл(путь)` — прочитать файл
- `записать_файл(путь, содержимое)` — записать в файл
- `существует_файл(путь)` — проверить существование

### JSON

- `json_разобрать(строка)` — разобрать JSON
- `json_создать(объект)` — создать JSON строку

### HTTP

- `http_сервер(порт, обработчик)` — создать HTTP сервер
- `http_получить(url)` — выполнить GET запрос

### Регулярные выражения

- `regex_найти(строка, шаблон)` — найти совпадения
- `regex_заменить(строка, шаблон, замена)` — заменить по шаблону
- `regex_совпадает(строка, шаблон)` — проверить совпадение

## Модульная система

Создайте файл `math.ya`:

```
экспорт конст пи = 3.14159

экспорт функция квадрат(x)
    возврат x * x
конец

экспорт функция куб(x)
    возврат x * x * x
конец
```

Используйте в другом файле:

```
импорт математика из "math.ya"

вывод(математика.пи)         # 3.14159
вывод(математика.квадрат(5)) # 25
вывод(математика.куб(3))     # 27
```

С алиасом:

```
импорт математика из "math.ya" как м

вывод(м.квадрат(10))         # 100
```

## Редактор кода

Расширение для VS Code и VSCodium с подсветкой синтаксиса, автодополнением и сниппетами.

### Установка

Скачайте файл `yasniy-lang-0.42.0.vsix` из репозитория и установите:

```bash
code --install-extension yasniy-lang-0.42.0.vsix
```

Или для VSCodium:

```bash
codium --install-extension yasniy-lang-0.42.0.vsix
```

### Возможности

- Подсветка синтаксиса для всех конструкций языка
- Автодополнение для 62 встроенных функций
- Сниппеты для быстрого написания кода
- Команды для запуска программ (Cmd+Shift+R)

### Сниппеты

Начните вводить ключевое слово и нажмите Tab:

- `функция` → шаблон функции
- `если` → условный оператор
- `для` → цикл
- `класс` → шаблон класса
- `вывод` → вывод с параметром
- `фильтр` → фильтрация массива

## Разработка

Проект написан на Go. Структура:

- `main.go` — точка входа
- `lexer/` — лексический анализатор
- `parser/` — синтаксический анализатор
- `ast/` — абстрактное синтаксическое дерево
- `interpreter/` — интерпретатор

Для запуска тестов:

```bash
go test ./...
```

Для сборки:

```bash
go build -o yasniy main.go
```
