## Основы синтаксиса

### Комментарии

Однострочные комментарии начинаются с символа `#`:

```
# Это комментарий
вывод("Привет")  # Комментарий в конце строки
```

### Точка с запятой

Точка с запятой не требуется. Каждая инструкция пишется на новой строке.

### Блоки кода

Блоки кода завершаются ключевым словом `конец`:

```
если условие
    # код блока
конец
```

## Типы данных

### Целые числа

```
конст возраст = 25
конст отрицательное = -10
```

### Дробные числа

```
конст пи = 3.14159
конст температура = -5.5
```

### Строки

Строки можно записывать в двойных или одинарных кавычках:

```
конст имя = "Иван"
конст фамилия = 'Петров'
```

**Интерполяция работает только в двойных кавычках:**

```
конст имя = "Мария"
вывод("Привет, {имя}!")           # Привет, Мария!
вывод('Привет, {имя}!')           # Привет, {имя}! (без интерполяции)

конст x = 5
конст y = 10
вывод("Результат: {x + y}")       # Результат: 15
```

### Булевы значения

```
конст истина = да
конст ложь = нет
```

### Массивы

```
конст числа = [1, 2, 3, 4, 5]
конст смешанный = [1, "текст", да, [1, 2]]

# Trailing comma разрешена
конст список = [
    элемент1,
    элемент2,
    элемент3,  # запятая после последнего элемента
]
```

### Объекты

```
конст человек = {
    "имя": "Иван",
    "возраст": 25,
    "город": "Москва"
}
```

## Переменные

### Неизменяемые переменные

Объявляются с помощью `конст`:

```
конст имя = "Иван"
# имя = "Петр"  # Ошибка: нельзя изменить
```

### Изменяемые переменные

Объявляются с помощью `перем`:

```
перем счетчик = 0
счетчик = 1
счетчик = счетчик + 1
```

## Операторы

### Арифметические операторы

```
конст сумма = 5 + 3        # 8
конст разность = 10 - 4    # 6
конст произведение = 3 * 4 # 12
конст частное = 15 / 3     # 5
конст остаток = 17 % 5     # 2
```

### Операторы сравнения

```
5 == 5    # да (равно)
5 != 3    # да (не равно)
5 > 3     # да (больше)
5 < 10    # да (меньше)
5 >= 5    # да (больше или равно)
3 <= 5    # да (меньше или равно)
```

### Логические операторы

```
да и да       # да
да и нет      # нет
да или нет    # да
нет или нет   # нет
не да         # нет
не нет        # да
```

### Тернарный оператор

```
конст возраст = 20
конст статус = возраст >= 18 ? "взрослый" : "ребенок"
вывод(статус)  # взрослый
```

```
класс Человек

        это.возраст = возраст
```

```
конст человек = новый Человек("Иван", 25)
```

```
вывод(человек.имя)      # Иван

человек.возраст = 26
вывод(человек.возраст)  # 26
```

### Наследование

```
класс Студент наследует Человек

конст студент = новый Студент("Мария", 20, "МГУ")
студент.представиться()  # Меня зовут Мария, мне 20 лет
студент.учиться()        # Мария учится в МГУ
```

## Массивы

### Создание массива

```
конст пустой = []
конст числа = [1, 2, 3, 4, 5]
конст смешанный = [1, "текст", да]
```

### Доступ к элементам

```
конст числа = [10, 20, 30, 40, 50]
вывод(числа[0])   # 10
вывод(числа[2])   # 30
вывод(числа[-1])  # 50 (последний элемент)
```

### Изменение элементов

```
перем числа = [1, 2, 3]
числа[1] = 10
вывод(числа)  # [1, 10, 3]
```

### Операции с массивами

```
конст числа = [1, 2, 3]

# Добавление элемента
добавить(числа, 4)
вывод(числа)  # [1, 2, 3, 4]

# Удаление элемента
удалить(числа, 1)
вывод(числа)  # [1, 3, 4]

# Вставка элемента
вставить(числа, 1, 2)
вывод(числа)  # [1, 2, 3, 4]

# Длина массива
вывод(длина(числа))  # 4
```

### Методы массивов

```
конст числа = [1, 2, 3, 4, 5]

# Фильтрация
конст четные = фильтр(числа, х => х % 2 == 0)
вывод(четные)  # [2, 4]

# Преобразование
конст удвоенные = преобразовать(числа, х => х * 2)
вывод(удвоенные)  # [2, 4, 6, 8, 10]

# Поиск
конст найденное = найти(числа, х => х > 3)
вывод(найденное)  # 4

# Проверка всех элементов
вывод(все(числа, х => х > 0))  # да

# Проверка хотя бы одного
вывод(любой(числа, х => х > 10))  # нет

# Сумма
вывод(сумма(числа))  # 15

# Сортировка
конст несортированный = [3, 1, 4, 1, 5]
вывод(сортировать(несортированный))  # [1, 1, 3, 4, 5]

# Сортировка с компаратором
класс Задача
    функция инициализация(название, приоритет)
        это.название = название
        это.приоритет = приоритет
    конец
конец

конст задачи = []
задачи.добавить(Задача("Кофе", "низкий"))
задачи.добавить(Задача("Учеба", "высокий"))
задачи.добавить(Задача("Работа", "средний"))

# Сортировка по приоритету (высокий > средний > низкий)
задачи.сортировать(функция(а, б)
    конст вес = функция(п)
        конст результат = совпадение п
            когда "высокий": 0
            когда "средний": 1
            когда "низкий": 2
            иначе: 3
        конец
        возврат результат
    конец
    возврат вес(а.приоритет) > вес(б.приоритет)
конец)

# Разворот
вывод(реверс(числа))  # [5, 4, 3, 2, 1]
```

## Строки

### Операции со строками

```
конст текст = "Привет, мир!"

# Длина строки
вывод(длина(текст))  # 12

# Верхний регистр
вывод(верхний(текст))  # ПРИВЕТ, МИР!

# Нижний регистр
вывод(нижний(текст))  # привет, мир!

# Подстрока
вывод(подстрока(текст, 0, 6))  # Привет

# Замена
вывод(заменить(текст, "мир", "Ясный"))  # Привет, Ясный!

# Разделение
конст слова = разделить(текст, " ")
вывод(слова)  # ["Привет,", "мир!"]

# Соединение
конст массив = ["один", "два", "три"]
вывод(соединить(массив, ", "))  # один, два, три

# Обрезка пробелов
вывод(обрезать("  текст  "))  # текст

# Проверки
вывод(начинается_с("привет", "при"))  # да
вывод(заканчивается_на("привет", "ет"))  # да
вывод(содержит("привет", "иве"))  # да

# Повторение
вывод(повторить("ха", 3))  # хахаха
```

## Встроенные функции

### Ввод/вывод

**вывод(значение)** — вывести значение в консоль

```
вывод("Привет")
вывод(42)
вывод([1, 2, 3])
```

**ввод(приглашение)** — получить ввод от пользователя

```
конст имя = ввод("Введите имя: ")
вывод("Привет, {имя}!")

конст возраст = число(ввод("Введите возраст: "))
```

### Работа с типами

**тип(значение)** — получить тип значения

```
вывод(тип(42))        # INTEGER
вывод(тип(3.14))      # FLOAT
вывод(тип("текст"))   # STRING
вывод(тип([1, 2]))    # ARRAY
```

**строка(значение)** — преобразовать в строку

```
вывод(строка(42))     # "42"
вывод(строка(да))     # "да"
```

**число(строка)** — преобразовать в число

```
вывод(число("42"))    # 42
вывод(число("3.14"))  # 3.14
```

### Математические функции

**мин(a, b)** — минимум из двух чисел

```
вывод(мин(5, 10))  # 5
```

**макс(a, b)** — максимум из двух чисел

```
вывод(макс(5, 10))  # 10
```

**степень(основание, показатель)** — возведение в степень

```
вывод(степень(2, 3))  # 8
```

**корень(число)** — квадратный корень

```
вывод(корень(16))  # 4
```

**абс(число)** — абсолютное значение

```
вывод(абс(-5))  # 5
```

**округл(число)** — округление

```
вывод(округл(3.7))  # 4
вывод(округл(3.2))  # 3
```

**случайное()** — случайное число от 0 до 1

```
конст случ = случайное()
вывод(случ)  # например, 0.7234
```

### Работа с файлами

**читать_файл(путь)** — прочитать содержимое файла

```
конст содержимое = читать_файл("test.txt")
вывод(содержимое)
```

**записать_файл(путь, содержимое)** — записать в файл

```
записать_файл("output.txt", "Привет, мир!")
```

**существует_файл(путь)** — проверить существование файла

```
если существует_файл("config.txt")
    вывод("Файл существует")
иначе
    вывод("Файл не найден")
конец
```

### Регулярные выражения

**regex_найти(строка, шаблон)** — найти совпадения

```
конст текст = "Телефон: 123-456-7890"
конст номер = regex_найти(текст, "\\d{3}-\\d{3}-\\d{4}")
вывод(номер)  # ["123-456-7890"]
```

**regex_заменить(строка, шаблон, замена)** — заменить по шаблону

```
конст текст = "Привет, мир!"
конст результат = regex_заменить(текст, "мир", "Ясный")
вывод(результат)  # Привет, Ясный!
```

**regex_совпадает(строка, шаблон)** — проверить совпадение

```
конст email = "test@example.com"
конст валидный = regex_совпадает(email, "^[\\w.-]+@[\\w.-]+\\.\\w+$")
вывод(валидный)  # да
```

## Продвинутые возможности

### Деструктуризация массивов

```
конст [a, b, c] = [1, 2, 3]
вывод(a)  # 1
вывод(b)  # 2
вывод(c)  # 3
```

С rest оператором:

```
конст [первый, второй, ...остальные] = [1, 2, 3, 4, 5]
вывод(первый)      # 1
вывод(второй)      # 2
вывод(остальные)   # [3, 4, 5]
```

### Деструктуризация объектов

```
конст человек = {"имя": "Иван", "возраст": 25, "город": "Москва"}
конст {"имя": имя, "возраст": возраст} = человек
вывод(имя)      # Иван
вывод(возраст)  # 25
```

### Spread оператор

В массивах:

```
конст массив1 = [1, 2, 3]
конст массив2 = [4, 5, 6]
конст объединенный = [...массив1, ...массив2]
```
