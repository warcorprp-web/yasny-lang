# Синтаксис языка "Простой"

## Ключевые слова

### Объявления
- `пусть` — неизменяемая переменная
- `var` — изменяемая переменная
- `функция` — объявление функции
- `класс` — объявление класса
- `данные` — data class (короткая форма)

### Управление потоком
- `если` — условие
- `иначеесли` — else if
- `иначе` — else
- `проверка` — guard statement
- `выбор` — pattern matching
- `когда` — case в pattern matching

### Циклы
- `для` — for loop
- `от` — from (в циклах)
- `до` — to (в циклах)
- `в` — in (для итерации)
- `пока` — while loop
- `повтори` — repeat N times

### Функции
- `вернуть` — return
- `асинх` — async function
- `ждать` — await

### Обработка ошибок
- `попытка` — try
- `поймать` — catch
- `результат` — Result type
- `ошибка` — Error type
- `успех` — Success (для Result)

### Модули
- `импорт` — import
- `из` — from (в импортах)
- `как` — as (алиас)
- `экспорт` — export

### Специальные значения
- `да` — true
- `нет` — false/null/none
- `это` — this/self

### Операторы
- `и` — and
- `или` — or
- `не` — not

## Типы данных

### Примитивные
- `целое` — integer
- `дробное` — float
- `строка` — string
- `булево` — boolean

### Коллекции
- `список` — list/array
- `словарь` — dictionary/map
- `множество` — set

### Специальные
- `ошибка?` — optional error
- `строка?` — optional type (может быть нет)

## Синтаксис

### Переменные
```
пусть имя = "Вася"
var возраст = 15
пусть число: целое = 42
```

### Функции
```
функция привет(имя: строка)
    вывод "Привет, {имя}!"
конец

функция сумма(а: целое, б: целое) -> целое
    вернуть а + б
конец

// Короткая форма
пусть умножить = (а, б) => а * б
```

### Условия
```
если возраст >= 18
    вывод "Взрослый"
иначеесли возраст >= 13
    вывод "Подросток"
иначе
    вывод "Ребенок"
конец
```

### Циклы
```
для i от 1 до 10
    вывод i
конец

для элемент в список
    вывод элемент
конец

пока условие
    // код
конец

повтори 5 раз
    вывод "Привет!"
конец
```

### Коллекции
```
пусть числа = [1, 2, 3, 4, 5]
пусть человек = {"имя": "Вася", "возраст": 25}
```

### Классы
```
класс Человек
    пусть имя: строка
    var возраст: целое
    
    функция инит(имя: строка, возраст: целое)
        это.имя = имя
        это.возраст = возраст
    конец
    
    функция представиться()
        вывод "Я {это.имя}"
    конец
конец
```

## Встроенные функции

### Ввод/вывод
- `вывод(...)` — print
- `ввод(подсказка)` — input

### Математика
- `округл(число)` — round
- `случайное(от, до)` — random
- `мин(а, б)` — min
- `макс(а, б)` — max

### Строки
- `длина(строка)` — length
- `часть(строка, начало, конец)` — substring
- `заменить(строка, старое, новое)` — replace

### Коллекции
- `добавить(список, элемент)` — append
- `удалить(список, индекс)` — remove
- `сортировать(список)` — sort

### Графика
- `холст(ширина, высота)` — create canvas
- `круг(х, у, радиус, цвет)` — draw circle
- `линия(х1, у1, х2, у2, цвет)` — draw line
- `текст(х, у, текст, размер)` — draw text
- `показать()` — show canvas
