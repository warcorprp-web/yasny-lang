# Шпаргалка по Ясному

Краткий справочник по синтаксису для быстрого использования. Полное
описание — в README.md.

## Базовые правила

- Расширение файла: `.ya`.
- Комментарий: только `#`. Многострочного комментария нет.
- Точка с запятой не нужна.
- Каждый блок открывается ключевым словом и закрывается `конец`.
- Можно писать тело блока в одну строку через двоеточие:
  `если x > 0: вывод(x)`.
- Строки в двойных кавычках поддерживают интерполяцию: `"{x + 1}"`.

## Переменные

```
конст имя = "Аня"          # неизменяемая (основная форма)
перем счётчик = 0          # изменяемая

счётчик = счётчик + 1
счётчик += 5
```

## Типы данных

```
конст число = 42
конст дробь = 3.14
конст текст = "строка"
конст флаг = да           # или нет
конст пустота = ничего    # null-значение
конст массив = [1, 2, 3]
конст словарь = {"имя": "Иван", "возраст": 30}
```

Проверка на ничего:

```
если значение == ничего: вывод("ничего нет")
```

## Условия

```
если условие
    ...
иначеесли другое
    ...
иначе
    ...
конец
```

`если` — это выражение, может возвращать значение:

```
конст x = если y > 0: y иначе: -y
```

    ...
конец

для элемент в коллекция
    ...
конец

пока условие
    ...

## Функции

```
функция плюс(a, b)
    вернуть a + b
конец

функция степень(основание, показатель = 2): основание * основание

функция сумма_всех(...числа)
    перем итог = 0
    для n в числа: итог += n
    вернуть итог
конец

конст удвоить = x => x * 2
```

Соглашение для предикатов: имя функции, возвращающей `да`/`нет`,
может заканчиваться на `?`:

```
функция чётное?(n): n % 2 == 0
функция простое?(n)
    если n < 2: вернуть нет
    # ...
    вернуть да
конец

вывод(чётное?(4))    # да
```

## Классы

```
класс Точка
    функция создать(x, y)
        это.x = x
        это.y = y
    конец

    функция длина()
        вернуть корень(это.x * это.x + это.y * это.y)
    конец
конец

конст p = Точка(3, 4)
вывод(p.длина())
```

Наследование:

```
класс Кот наследует Животное
    функция создать(имя, цвет)
        родитель.создать(имя)
        это.цвет = цвет
    конец
конец
```

`создать` — имя конструктора. `это` — текущий экземпляр.
`родитель` — родительский класс.

## Современные конструкции

```
# Цепочка вычислений (pipe)
числа |> фильтр(x => x > 0) |> сумма

# Optional chaining
пользователь?.адрес?.город ?? "не указан"

# Spread
конст оба = [...первый, ...второй]

# Деструктуризация
конст [a, b, ...остальное] = массив

# List comprehension
конст квадраты = [x * x для x в [1, 2, 3, 4] если x % 2 == 0]

# Срезы
числа[1:4]    # с 1 по 3 включительно
числа[:3]     # первые три
числа[-2:]    # последние два
```

```

Индексация и сравнение:

```
конст с = "привет"
вывод(с[0])           # п
вывод(с[-1])          # т (отрицательный индекс с конца)
вывод(длина(с))       # 6 (символы, не байты)

вывод("а" < "б")      # да (лексикографическое сравнение)
вывод("я" > "а")      # да
```

### Числа

```

## Модули

Файл `модуль.ya`:

```
экспорт функция приветствие(имя)
    вернуть "Привет, {имя}!"
конец
```

Использование:

```
импорт м из "модуль.ya"
вывод(м.приветствие("Аня"))
```

## Пакетный менеджер

Команды:

```
yasny инит [имя]                    # создать пакет.json и главный.ya
yasny подключить ИМЯ                # установить по короткому имени из реестра
yasny подключить URL[@версия]       # установить по полному URL
yasny подключить                    # установить всё из манифеста
yasny удалить ИМЯ                   # удалить пакет
yasny список                        # показать зависимости
yasny запустить [файл]              # запустить точку входа
yasny формат файл.ya                # отформатировать (в stdout)
yasny формат -в файл.ya             # отформатировать и переписать
yasny проверить файл.ya             # линтер (статический анализ)
yasny проверить .                   # проверить все .ya рекурсивно
yasny помощь                        # справка
```

Самый простой сценарий — один файл и пакет:

```
echo 'импорт м из "матем"
вывод(м.факториал(5))' > программа.ya

yasny подключить матем     # пакет.json создастся сам
yasny программа.ya
```

Манифест `пакет.json` и lock-файл `пакет.lock` создаются автоматически.

Реестр коротких имён: [yasny-registry](https://github.com/warcorprp-web/yasny-registry).
Свой пакет можно добавить через PR.

Пакеты ставятся в папку `пакеты/имя/`, импорт по короткому имени:

```
импорт м из "матем"
вывод(м.факториал(5))
```

## Форматирование

```
yasny формат файл.ya       # вывести в stdout
yasny формат -в файл.ya    # переписать файл
yasny формат -в .          # все .ya рекурсивно
```

Форматер приводит код к канону из [СТИЛЬ.md](../СТИЛЬ.md):
- 4 пробела отступа
- пробелы вокруг операторов и после запятых
- одна пустая строка между функциями верхнего уровня
- trailing comma в многострочных литералах

Сохраняет: комментарии, inline/блочные формы, лямбды, pipeline,
декораторы, скобки в выражениях. Поведение программы не меняется.

## Чего в языке нет

- Аннотаций типов (язык динамический).
- `++`, `--` (используйте `+= 1`, `-= 1`).
- Нескольких видов комментариев. Только `#`.
- `повтори N раз` (используйте `для i от 1 до N`).
- Старых форм: `пусть`, `var`, `процедура`, `возврат`, `истина`,
  `ложь`, `//` как комментарий. Они удалены — используйте канонические.
