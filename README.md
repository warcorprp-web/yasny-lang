# Ясный

Язык программирования на русском языке для обучения основам программирования.

## Зачем нужен Ясный?

### Проблема: когнитивная перегрузка на старте

Когда новичок без знания английского начинает учить программирование на Python или JavaScript, его мозг одновременно решает шесть задач:

1. Что означает `def`? (definition, но он не знает английского)
2. Что такое `return`?
3. Почему нужны двоеточия и отступы?
4. Как работает синтаксис?
5. Что делают встроенные функции?
6. **Как работает сам алгоритм?**

Только последний вопрос — про суть программирования. Остальные пять — про синтаксис и английский язык.

### Решение: разделить обучение на этапы

**Этап 1 (Ясный):** Изучить алгоритмическое мышление на родном языке
- Переменные, условия, циклы, функции, классы
- Понятные сообщения об ошибках с подсказками
- Фокус на логике, а не на синтаксисе

**Этап 2 (Python/JS):** Переключить "раскладку клавиатуры"
- Все концепции уже понятны
- Нужно только запомнить английские ключевые слова
- Переход занимает 1-2 недели вместо 2-3 месяцев

### Что показывают исследования

Научные данные по языкам программирования на родном языке:

- **Quorum** (США): новички делают на 40% меньше синтаксических ошибок, когда синтаксис интуитивно понятен
- **КуМир** (Россия): школьники, начавшие с русского языка, не отстают при переходе на Pascal/Python
- **なでしこ** (Япония): студенты демонстрируют лучшее понимание алгоритмов при последующем переходе на Java

**Вывод:** родной язык на старте не задерживает — он ускоряет обучение.

### Сравнение времени обучения

| Путь | Время до базового уровня | Время на переход |
|------|-------------------------|------------------|
| Сразу Python/JS | 4-5 месяцев | — |
| Ясный → Python/JS | 2-3 месяца | 2 недели |

Экономия: 2-3 месяца (целый семестр).

### Что переносится в Python/JS

Все концепции переносятся один-в-один:

| Концепция | Ясный | Python | Переносится? |
|-----------|-------|--------|--------------|
| Переменная | `конст x = 5` | `x = 5` | Да |
| Условие | `если ... конец` | `if ...:` | Да |
| Цикл | `пока ... конец` | `while ...:` | Да |
| Функция | `функция ... конец` | `def ...:` | Да |
| Массив | `[1, 2, 3]` | `[1, 2, 3]` | Да |
| Объект | `{"ключ": значение}` | `{"ключ": значение}` | Да |
| Класс | `класс ... конец` | `class ...:` | Да |

Меняются только ключевые слова (15 минут по таблице соответствий).

### Когда Ясный НЕ нужен

- Студент уже знает английский на уровне B1+
- Цель — быстрое трудоустройство (буткемп)
- Студент уже программировал на другом языке

Ясный — для первого соприкосновения с программированием у тех, для кого английский — дополнительный барьер.

## Первая программа

Создайте файл `hello.ya`:

```
вывод("Привет, мир!")
```

Запустите:

```bash
yasny hello.ya
```

Или используйте интерактивный режим (REPL):

```bash
yasny
>>> вывод("Привет!")
Привет!
>>> выход
```

## Основные возможности

### Переменные

```
конст имя = "Иван"           # Неизменяемая переменная
перем возраст = 25           # Изменяемая переменная
возраст = 26                 # Можно изменить
```

### Типы данных

```
конст число = 42             # Целое число
конст дробь = 3.14           # Дробное число
конст текст = "строка"       # Строка (двойные кавычки)
конст текст2 = 'строка'      # Строка (одинарные кавычки)
конст флаг = да              # Булево значение (да/нет)
конст список = [1, 2, 3]     # Массив
конст данные = {"ключ": "значение"}  # Объект
```

### Интерполяция строк

Работает только в двойных кавычках:

```
конст имя = "Мария"
вывод("Привет, {имя}!")      # Привет, Мария!
вывод('Привет, {имя}!')      # Привет, {имя}! (без интерполяции)

конст x = 5
конст y = 10
вывод("Сумма: {x + y}")      # Сумма: 15
```

### Функции

```
функция сумма(a, b)
    возврат a + b
конец

конст результат = сумма(5, 3)
вывод(результат)             # 8
```

Лямбда-функции:

```
конст удвоить = х => х * 2
вывод(удвоить(5))            # 10
```

### Условия

```
конст возраст = 18

если возраст >= 18
    вывод("Совершеннолетний")
иначеесли возраст >= 14
    вывод("Подросток")
иначе
    вывод("Ребенок")
конец
```

### Циклы

Цикл по массиву:

```
конст числа = [1, 2, 3, 4, 5]

для число в числа
    вывод(число)
конец
```

Цикл с условием:

```
перем счетчик = 0

пока счетчик < 5
    вывод(счетчик)
    счетчик = счетчик + 1
конец
```

Цикл с диапазоном:

```
для i от 1 до 10
    вывод(i)
конец
```

### Классы

```
класс Человек
    функция инициализация(имя, возраст)
        это.имя = имя
        это.возраст = возраст
    конец
    
    функция представиться()
        вывод("Меня зовут {это.имя}, мне {это.возраст} лет")
    конец
конец

конст человек = Человек("Иван", 25)
человек.представиться()
```

### Наследование

```
класс Животное
    функция инициализация(имя)
        это.имя = имя
    конец
    
    функция говорить()
        возврат "{это.имя} издает звук"
    конец
конец

класс Собака наследует Животное
    функция инициализация(имя, порода)
        родитель.инициализация(имя)
        это.порода = порода
    конец
    
    функция лаять()
        возврат "гав-гав"
    конец
конец

конст собака = Собака("Бобик", "Овчарка")
вывод(собака.говорить())  # Бобик издает звук
вывод(собака.лаять())     # гав-гав

# Если потомку не нужна своя инициализация - 
# конструктор родителя используется автоматически
класс Кот наследует Животное
    функция говорить()
        возврат "{это.имя} мяукает"
    конец
конец

конст барсик = Кот("Барсик")
вывод(барсик.говорить())  # Барсик мяукает
```

## Примеры кода

### Работа с массивами

```
конст числа = [1, 2, 3, 4, 5]

# Фильтрация
конст четные = фильтр(числа, х => х % 2 == 0)
вывод(четные)                # [2, 4]

# Преобразование
конст удвоенные = преобразовать(числа, х => х * 2)
вывод(удвоенные)             # [2, 4, 6, 8, 10]

# Поиск
конст найденное = найти(числа, х => х > 3)
вывод(найденное)             # 4

# Сумма
вывод(сумма(числа))          # 15
```

### Деструктуризация

```
конст [первый, второй, ...остальные] = [1, 2, 3, 4, 5]
вывод(первый)                # 1
вывод(второй)                # 2
вывод(остальные)             # [3, 4, 5]
```

### Spread оператор

```
конст массив1 = [1, 2, 3]
конст массив2 = [4, 5, 6]
конст объединенный = [...массив1, ...массив2]
вывод(объединенный)          # [1, 2, 3, 4, 5, 6]
```

### Optional chaining

```
конст данные = {"пользователь": {"имя": "Иван"}}

вывод(данные?.пользователь?.имя)     # Иван
вывод(данные?.адрес?.город)          # нет (не ошибка)
```

### Pattern matching

```
конст день = 3

конст название = совпадение день
    когда 1: "Понедельник"
    когда 2: "Вторник"
    когда 3: "Среда"
    когда 4: "Четверг"
    когда 5: "Пятница"
    иначе: "Выходной"
конец

вывод(название)              # Среда
```

### List comprehension

```
конст числа = [1, 2, 3, 4, 5]
конст квадраты = [x * x для x в числа если x % 2 == 0]
вывод(квадраты)              # [4, 16]
```

### Работа с файлами

```
# Запись в файл
записать_файл("test.txt", "Привет, мир!")

# Чтение из файла
конст содержимое = читать_файл("test.txt")
вывод(содержимое)

# Проверка существования
если существует_файл("test.txt")
    вывод("Файл существует")
конец
```

### HTTP сервер

```
функция обработчик(запрос)
    возврат {
        "статус": 200,
        "тело": "Привет из Ясного!"
    }
конец

http_сервер(8080, обработчик)
вывод("Сервер запущен на порту 8080")
```

### JSON

```
конст данные = {"имя": "Иван", "возраст": 25}

# Преобразовать в JSON
конст json = json_создать(данные)
вывод(json)                  # {"имя":"Иван","возраст":25}

# Разобрать JSON
конст объект = json_разобрать(json)
вывод(объект.имя)            # Иван
```

## Встроенные функции

### Ввод/вывод

- `вывод(значение)` — вывести в консоль
- `ввод(приглашение)` — получить ввод пользователя

### Работа с типами

- `тип(значение)` — получить тип значения
- `строка(значение)` — преобразовать в строку
- `число(строка)` — преобразовать в число
- `длина(массив)` — получить длину массива или строки

### Строки

- `разделить(строка, разделитель)` — разделить строку
- `соединить(массив, разделитель)` — соединить массив в строку
- `заменить(строка, старое, новое)` — заменить подстроку
- `верхний(строка)` — в верхний регистр
- `нижний(строка)` — в нижний регистр
- `подстрока(строка, начало, конец)` — получить подстроку
- `обрезать(строка)` — удалить пробелы с краев
- `начинается_с(строка, префикс)` — проверить начало
- `заканчивается_на(строка, суффикс)` — проверить конец
- `содержит(строка, подстрока)` — проверить наличие
- `повторить(строка, количество)` — повторить N раз

### Массивы

- `добавить(массив, элемент)` — добавить элемент
- `удалить(массив, индекс)` — удалить элемент
- `вставить(массив, индекс, элемент)` — вставить элемент
- `найти(массив, функция)` — найти элемент по условию
- `найти_индекс(массив, функция)` — найти индекс элемента
- `фильтр(массив, функция)` — отфильтровать массив
- `преобразовать(массив, функция)` — преобразовать элементы
- `дляКаждого(массив, функция)` — выполнить для каждого
- `свернуть(массив, начальное, функция)` — свернуть в значение
- `все(массив, функция)` — проверить все элементы
- `любой(массив, функция)` — проверить хотя бы один
- `сортировать(массив)` — отсортировать по умолчанию
- `сортировать(массив, функция)` — отсортировать с компаратором
- `реверс(массив)` — развернуть
- `объединить(массив1, массив2)` — объединить массивы
- `взять(массив, количество)` — взять N элементов
- `пропустить(массив, количество)` — пропустить N элементов
- `сумма(массив)` — сумма элементов
- `установить(массив)` — уникальные элементы

### Математика

- `мин(a, b)` — минимум
- `макс(a, b)` — максимум
- `степень(основание, показатель)` — возведение в степень
- `корень(число)` — квадратный корень
- `абс(число)` — абсолютное значение
- `округл(число)` — округление
- `случайное()` — случайное число от 0 до 1
- `диапазон(начало, конец)` — создать диапазон чисел

### Объекты

- `ключи(объект)` — получить ключи
- `значения(объект)` — получить значения

### Файлы

- `читать_файл(путь)` — прочитать файл
- `записать_файл(путь, содержимое)` — записать в файл
- `существует_файл(путь)` — проверить существование

### JSON

- `json_разобрать(строка)` — разобрать JSON
- `json_создать(объект)` — создать JSON строку

### HTTP

- `http_сервер(порт, обработчик)` — создать HTTP сервер
- `http_получить(url)` — выполнить GET запрос

### Регулярные выражения

- `regex_найти(строка, шаблон)` — найти совпадения
- `regex_заменить(строка, шаблон, замена)` — заменить по шаблону
- `regex_совпадает(строка, шаблон)` — проверить совпадение

## Модульная система

Создайте файл `math.ya`:

```
экспорт конст пи = 3.14159

экспорт функция квадрат(x)
    возврат x * x
конец

экспорт функция куб(x)
    возврат x * x * x
конец
```

Используйте в другом файле:

```
импорт математика из "math.ya"

вывод(математика.пи)         # 3.14159
вывод(математика.квадрат(5)) # 25
вывод(математика.куб(3))     # 27
```

С алиасом:

```
импорт математика из "math.ya" как м

вывод(м.квадрат(10))         # 100
```

## Переход на Python/JavaScript

После освоения Ясного переход на промышленные языки занимает 1-2 недели. Все концепции остаются теми же, меняются только ключевые слова.

### Таблица соответствий

#### Python

| Ясный | Python | Пример |
|-------|--------|--------|
| `конст` | (нет) | `x = 5` |
| `перем` | (нет) | `x = 5` |
| `функция` | `def` | `def имя():` |
| `возврат` | `return` | `return x` |
| `если` | `if` | `if x > 0:` |
| `иначе` | `else` | `else:` |
| `иначе_если` | `elif` | `elif x < 0:` |
| `пока` | `while` | `while x > 0:` |
| `для` | `for` | `for x in список:` |
| `класс` | `class` | `class Имя:` |
| `это` | `self` | `self.x = 5` |
| `да/нет` | `True/False` | `if True:` |
| `конец` | (отступы) | (4 пробела) |

#### JavaScript

| Ясный | JavaScript | Пример |
|-------|------------|--------|
| `конст` | `const` | `const x = 5;` |
| `перем` | `let` | `let x = 5;` |
| `функция` | `function` | `function имя() {}` |
| `возврат` | `return` | `return x;` |
| `если` | `if` | `if (x > 0) {}` |
| `иначе` | `else` | `else {}` |
| `иначе_если` | `else if` | `else if (x < 0) {}` |
| `пока` | `while` | `while (x > 0) {}` |
| `для` | `for` | `for (let x of список) {}` |
| `класс` | `class` | `class Имя {}` |
| `это` | `this` | `this.x = 5;` |
| `да/нет` | `true/false` | `if (true) {}` |
| `конец` | `}` | `}` |

### Пример перевода кода

**Ясный:**
```
функция факториал(n)
    если n <= 1
        возврат 1
    конец
    возврат n * факториал(n - 1)
конец

вывод(факториал(5))
```

**Python:**
```python
def факториал(n):
    if n <= 1:
        return 1
    return n * факториал(n - 1)

print(факториал(5))
```

**JavaScript:**
```javascript
function факториал(n) {
    if (n <= 1) {
        return 1;
    }
    return n * факториал(n - 1);
}

console.log(факториал(5));
```

## Разработка

Проект написан на Go. Структура:

- `main.go` — точка входа
- `lexer/` — лексический анализатор
- `parser/` — синтаксический анализатор
- `ast/` — абстрактное синтаксическое дерево
- `interpreter/` — интерпретатор

Для запуска тестов:

```bash
go test ./...
```

Для сборки:

```bash
go build -o yasny main.go
```
