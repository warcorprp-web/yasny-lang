# Шпаргалка по языку Ясный для AI

## Критически важно

1. **Комментарии**: только `#`, НЕ `//`
2. **Строки**: двойные `"текст"` или одинарные `'текст'` кавычки
3. **Интерполяция**: `"Привет, {имя}"` работает только в двойных кавычках
4. **Конкатенация**: `"Привет, " + имя` работает
5. **Классы**: `Студент("Иван")` работает БЕЗ `новый` (но `новый` тоже работает)
6. **Блоки**: всегда заканчиваются словом `конец`
7. **Trailing comma**: запятая после последнего элемента массива разрешена

## Синтаксис циклов

### Цикл по массиву
```yasny
для элемент в массив
    вывод(элемент)
конец
```

### Цикл с диапазоном
```yasny
для i от 1 до 10
    вывод(i)
конец

для i от 0 до 100 по 10
    вывод(i)  # 0, 10, 20, ...
конец
```

### Цикл while
```yasny
пока условие
    # код
конец
```

### НЕ ПОДДЕРЖИВАЕТСЯ
❌ `для перем i = 0; i < 10; i = i + 1` - НЕ РАБОТАЕТ
❌ `для (i = 0; i < 10; i++)` - НЕ РАБОТАЕТ

## Переменные

```yasny
конст имя = "Иван"           # Константа

```

## Составные операторы присваивания

```yasny

# Работает с массивами и объектами
массив[0] += 10
объект.счетчик += 1
```

## Встроенные функции (57 штук)

### Ввод/вывод
- `вывод(значение)` - вывести в консоль
- `ввод(приглашение)` - получить ввод

### Типы
- `тип(значение)` - получить тип
- `строка(значение)` - преобразовать в строку
- `число(строка)` - преобразовать в число

### Математика
- `мин(a, b)` - минимум
- `макс(a, b)` - максимум
- `степень(основание, показатель)` - возведение в степень
- `корень(число)` - квадратный корень
- `абс(число)` - абсолютное значение
- `округл(число)` - округление
- `случайное()` - случайное число 0-1
- `случайное(мин, макс)` - случайное целое число в диапазоне

### Массивы (базовые)
- `длина(массив)` - длина массива
- `добавить(массив, элемент)` - добавить в конец (изменяет массив)
- `удалить(массив, индекс)` - удалить по индексу (изменяет массив)
- `вставить(массив, индекс, элемент)` - вставить по индексу (изменяет массив)
- `диапазон(конец)` - создать массив [0, 1, ..., конец-1]
- `диапазон(начало, конец)` - создать массив [начало, ..., конец-1]
- `сортировать(массив)` - сортировать массив по умолчанию (изменяет массив)
- `сортировать(массив, функция)` - сортировать с компаратором (изменяет массив)
- `реверс(массив)` - развернуть массив (изменяет массив)
- `взять(массив, количество)` - взять первые N элементов
- `пропустить(массив, количество)` - пропустить первые N элементов
- `объединить(массив1, массив2, ...)` - объединить массивы
- `сумма(массив)` - сумма элементов массива чисел
- `соединить(массив, разделитель)` - объединить элементы в строку

### Методы массивов (вызываются через точку)
```yasny
массив.добавить(элемент)           # добавить в конец
массив.удалить(индекс)             # удалить по индексу
массив.вставить(индекс, элемент)   # вставить по индексу
массив.найти(функция)              # найти элемент
массив.фильтр(функция)             # отфильтровать
массив.карта(функция)              # преобразовать (map)
массив.свернуть(функция, начало)   # свернуть (reduce)
массив.все(функция)                # все элементы удовлетворяют
массив.любой(функция)              # хотя бы один удовлетворяет
массив.сортировать()               # сортировать
массив.реверс()                    # развернуть
массив.объединить(разделитель)     # объединить в строку
```

### Функции высшего порядка
- `найти(массив, функция)` - найти элемент по условию
- `найти_индекс(массив, функция)` - найти индекс элемента
- `фильтр(массив, функция)` - отфильтровать элементы
- `преобразовать(массив, функция)` - преобразовать элементы (map)
- `свернуть(массив, функция, начальное)` - свернуть массив (reduce)
- `дляКаждого(массив, функция)` - выполнить для каждого элемента
- `все(массив, функция)` - все элементы удовлетворяют условию
- `любой(массив, функция)` - хотя бы один удовлетворяет

### Строки
- `верхний(строка)` - в верхний регистр
- `нижний(строка)` - в нижний регистр
- `разделить(строка, разделитель)` - разделить на массив
- `заменить(строка, старое, новое)` - заменить подстроку
- `содержит(строка, подстрока)` - проверить наличие
- `начинается_с(строка, префикс)` - проверить начало
- `заканчивается_на(строка, суффикс)` - проверить конец
- `обрезать(строка)` - убрать пробелы по краям
- `подстрока(строка, начало, конец)` - вырезать подстроку
- `повторить(строка, количество)` - повторить строку N раз
- `найти(строка, подстрока)` - найти индекс подстроки

### Объекты (словари)
- `ключи(объект)` - получить массив ключей
- `значения(объект)` - получить массив значений

### Файлы
- `читать_файл(путь)` - прочитать файл
- `записать_файл(путь, содержимое)` - записать в файл
- `существует_файл(путь)` - проверить существование

### JSON
- `json_разобрать(строка)` - распарсить JSON строку в объект
- `json_создать(объект)` - преобразовать объект в JSON строку

### HTTP
- `http_получить(url)` - GET запрос, возвращает строку
- `http_сервер(порт, обработчик)` - создать HTTP сервер

### Регулярные выражения
- `regex_совпадает(строка, паттерн)` - проверить совпадение
- `regex_найти(строка, паттерн)` - найти все совпадения
- `regex_заменить(строка, паттерн, замена)` - заменить по паттерну

### Утилиты
- `ошибка(сообщение)` - выбросить ошибку
- `установить(экземпляр, ключ, значение)` - установить свойство экземпляра
- `загрузить(путь)` - загрузить и выполнить файл

## Операторы

### Арифметические
- `+` сложение
- `-` вычитание
- `*` умножение
- `/` деление
- `%` остаток от деления

### Сравнение
- `==` равно
- `!=` не равно
- `<` меньше
- `>` больше
- `<=` меньше или равно
- `>=` больше или равно

### Логические
- `и` логическое И
- `или` логическое ИЛИ
- `не` логическое НЕ

### Специальные
- `..` диапазон: `1..10`
- `...` spread: `[1, ...массив, 5]`
- `?.` optional chaining: `объект?.поле?.метод()`

## Константы

- `истина` / `да` - true
- `ложь` / `нет` - false
- `нуль` - null

## Важные замечания для AI

1. **Всегда используй синтаксис из этой шпаргалки**
2. **Проверяй названия функций** - они на русском и могут отличаться от ожидаемых
3. **Цикл for** работает только как `для элемент в массив` или `для i от 1 до 10`
4. **Массивы нельзя складывать** - используй `.добавить()`
5. **Классы создаются без `новый`** (но `новый` тоже работает для совместимости)
6. **Интерполяция работает**: `"Привет, {имя}"` корректна
7. **Методы массивов** вызываются через точку: `массив.карта(...)`, `массив.фильтр(...)`
8. **Свертка (reduce)**: первый аргумент - функция, второй - начальное значение
