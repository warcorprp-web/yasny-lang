# Ясный - Расширение для VS Code

Полная поддержка языка программирования Ясный для Visual Studio Code и VSCodium.

## Возможности

- **Подсветка синтаксиса** для всех конструкций языка
- **100+ сниппетов** для быстрого написания кода
- **Автодополнение** для всех встроенных функций
- **Команды запуска** программ (Cmd+Shift+R)
- **Поддержка всех возможностей:**
  - Try-catch-finally (попытка-поймать-всегда)
  - Spread оператор (...)
  - Optional chaining (?.)
  - Деструктуризация массивов и объектов
  - Pattern matching (совпадение-когда)
  - Классы с наследованием
  - Модульная система (импорт-экспорт)
  - HTTP сервер
  - Работа с файлами и JSON
  - Регулярные выражения

## Установка

### Из файла VSIX

```bash
code --install-extension yasny-lang-0.44.0.vsix
```

Или для VSCodium:

```bash
codium --install-extension yasny-lang-0.44.0.vsix
```

### Через интерфейс

1. Откройте VS Code
2. Нажмите `Cmd+Shift+P` (macOS) или `Ctrl+Shift+P` (Windows/Linux)
3. Введите "Extensions: Install from VSIX"
4. Выберите файл `yasny-lang-0.44.0.vsix`

## Использование

### Создание файла

Создайте файл с расширением `.ya`:

```yasny
вывод("Привет, мир!")
```

### Запуск программы

- **Cmd+Shift+R** (macOS) или **Ctrl+Shift+R** (Windows/Linux) - запустить в панели вывода
- **Cmd+Alt+R** (macOS) или **Ctrl+Alt+R** (Windows/Linux) - запустить в терминале

### Автодополнение

Начните вводить название функции и нажмите `Tab`:

- `вывод` → `вывод(значение)`
- `функция` → шаблон функции
- `если` → условный оператор
- `для` → цикл
- `фильтр` → `фильтр(массив, x => x > 0)`

### Сниппеты (100+)

Доступны сниппеты для всех конструкций языка. Начните вводить и нажмите Tab:

**Управляющие конструкции:**
- `функция` - объявление функции
- `если` / `иначе` - условные операторы
- `для` - цикл for-in
- `пока` - цикл while
- `совпадение` - pattern matching
- `попытка` - try-catch
- `попытка-всегда` - try-catch-finally
- `бросить` - throw ошибку

**Классы и ООП:**
- `класс` - объявление класса
- `класс-наследует` - класс с наследованием

**Модули:**
- `импорт` - импорт модуля
- `экспорт` - экспорт значения

**Современные возможности:**
- `spread` - spread оператор (...)
- `optional` - optional chaining (?.)
- `деструкт-массив` - деструктуризация массива с rest
- `деструкт-объект` - деструктуризация объекта
- `массив-trailing` - массив с trailing comma
- `лямбда` - лямбда-функция
- `listcomp` - list comprehension

**Работа с данными:**
- `сортировать-компаратор` - сортировка с функцией сравнения
- `фильтр` - фильтрация массива
- `преобразовать` - map для массива
- `найти` - поиск элемента

**Файлы и сеть:**
- `читать-файл` - чтение файла с обработкой ошибок
- `json-парсинг` - парсинг JSON
- `regex-найти` - поиск по регулярному выражению
- `http-сервер` - HTTP сервер с обработчиком

**Встроенные функции (60+):**
- `вывод`, `ввод`, `длина`, `тип`, `строка`, `число`
- `разделить`, `соединить`, `заменить`, `верхний`, `нижний`
- `добавить`, `удалить`, `вставить`, `сортировать`, `реверс`
- `фильтр`, `преобразовать`, `найти`, `сумма`, `мин`, `макс`
- И многие другие...

## Подсветка синтаксиса

Расширение поддерживает подсветку для:

- Ключевых слов: `если`, `для`, `функция`, `класс`, `импорт`, `экспорт`
- Встроенных функций: `вывод`, `ввод`, `фильтр`, `преобразовать`, и т.д.
- Операторов: `+`, `-`, `*`, `/`, `==`, `!=`, `...`, `?.`
- Строк с интерполяцией: `"Привет, {имя}!"`
- Комментариев: `# комментарий`
- Чисел и булевых значений

## Настройки

Путь к интерпретатору можно настроить в settings.json:

```json
{
  "yasny.interpreterPath": "/usr/local/bin/yasny"
}
```

## Примеры кода

### Простая программа

```yasny
конст имя = "Мир"
вывод("Привет, {имя}!")
```

### Функция

```yasny
функция сумма(a, b)
    возврат a + b
конец

вывод(сумма(5, 3))
```

### Работа с массивами

```yasny
конст числа = [1, 2, 3, 4, 5]
конст четные = фильтр(числа, х => х % 2 == 0)
вывод(четные)
```

### Модули

```yasny
# math.ya
экспорт функция квадрат(x)
    возврат x * x
конец

# main.ya
импорт математика из "math.ya"
вывод(математика.квадрат(5))
```
