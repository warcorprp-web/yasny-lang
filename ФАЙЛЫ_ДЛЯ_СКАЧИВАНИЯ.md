# 📦 Файлы для установки на macOS

## Что нужно скачать:

### 1️⃣ Интерпретатор (выбери один):
- **prostoy-macos-arm64** (11 MB) - для Apple Silicon (M1/M2/M3)
- **prostoy-macos-amd64** (10 MB) - для Intel Mac

Находятся в папке: `dist/`

### 2️⃣ VS Code расширение:
Вся папка: `vscode-extension/`

Содержит:
- `package.json` - манифест расширения
- `language-configuration.json` - конфигурация языка
- `syntaxes/prostoy.tmLanguage.json` - подсветка синтаксиса
- `snippets/prostoy.json` - сниппеты кода
- `src/extension.ts` - код расширения
- `tsconfig.json` - настройки TypeScript
- `README.md` - документация

### 3️⃣ Документация:
- `РУКОВОДСТВО.md` - полное руководство по языку
- `УСТАНОВКА_НА_MAC.md` - инструкция по установке
- `README.md` - общая информация

## 🚀 Быстрая установка:

```bash
# 1. Скачай архив prostoy-macos-package.tar.gz (11 MB)

# 2. Распакуй
tar -xzf prostoy-macos-package.tar.gz

# 3. Установи интерпретатор
cd prostoy-lang
sudo cp dist/prostoy-macos-arm64 /usr/local/bin/prostoy  # для M1/M2/M3
# или
sudo cp dist/prostoy-macos-amd64 /usr/local/bin/prostoy  # для Intel
sudo chmod +x /usr/local/bin/prostoy

# 4. Проверь
prostoy
>>> вывод("Работает!")
>>> выход

# 5. Установи расширение для VSCodium
cd vscode-extension
npm install
npm run compile
npm run package
codium --install-extension prostoy-lang-0.40.0.vsix

# 6. Открой VSCodium и создай файл test.pr
```

## 📝 Тестовый файл (test.pr):

```prostoy
вывод("=== Тест языка Простой ===")

# Деструктуризация
конст [a, b, c] = [1, 2, 3]
вывод("a={a}, b={b}, c={c}")

# Spread
конст массив1 = [1, 2, 3]
конст массив2 = [4, 5, 6]
конст все = [...массив1, ...массив2]
вывод("Spread: {строка(все)}")

# Optional chaining
конст obj = {"поле": {"вложенное": "значение"}}
вывод("Optional: {obj?.поле?.вложенное}")

# Match
конст день = 3
конст название = совпадение день
    когда 1: "Пн"
    когда 2: "Вт"
    когда 3: "Ср"
    иначе: "Другое"
конец
вывод("Match: {название}")

# Методы
вывод("Методы: {'привет'.начинается_с('при')}")

конст числа = [1, 2, 3, 4, 5]
вывод("Найти: {числа.найти(х => х > 3)}")

вывод("\n✅ Все работает!")
```

## 🎯 Горячие клавиши в VSCodium:

- **Cmd+Shift+R** - запустить программу
- **Cmd+Alt+R** - запустить в терминале
- Начни вводить `функция` и нажми Tab - автодополнение!

## ❓ Проблемы?

Если не запускается:
```bash
# Проверь права
ls -l /usr/local/bin/prostoy

# Должно быть: -rwxr-xr-x

# Если нет, исправь:
sudo chmod +x /usr/local/bin/prostoy

# Проверь PATH
echo $PATH | grep /usr/local/bin

# Если нет, добавь в ~/.zshrc:
echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```
