## Работа с числами

### Сумма от 1 до N

```
функция сумма_до(n)
    перем итог = 0
    для i от 1 до n: итог += i
    вернуть итог
конец

вывод(сумма_до(100))    # 5050
```

### Чётное или нечётное

```
функция чётное(n): n % 2 == 0

вывод(чётное(4))    # да
вывод(чётное(7))    # нет
```

### Среднее арифметическое

```
функция среднее(числа)
    если длина(числа) == 0: вернуть 0
    вернуть сумма(числа) / длина(числа)
конец

вывод(среднее([10, 20, 30, 40]))    # 25
```

### Округление до N знаков

```
функция округлить_до(x, знаков)
    конст множитель = степень(10, знаков)
    вернуть округл(x * множитель) / множитель
конец

вывод(округлить_до(3.14159, 2))    # 3.14
```

### Случайное число в диапазоне

```
вывод(случайное.целое(1, 100))     # целое 1..100
вывод(случайное.число())           # дробное 0..1
```

## Работа со строками

### Перевернуть строку

```
функция перевернуть(s)
    перем результат = ""
    для i от длина(s) - 1 до 0 по -1
        результат += s[i]
    конец
    вернуть результат
конец

вывод(перевернуть("Привет"))    # тевирП
```

### Подсчёт слов

```
функция количество_слов(текст)
    вернуть длина(разделить(обрезать(текст), " "))
конец

вывод(количество_слов("Это предложение из пяти слов"))    # 5
```

### Замена нескольких подстрок

```
функция заменить_многое(текст, замены)
    перем результат = текст
    для пара в замены
        результат = заменить(результат, пара[0], пара[1])
    конец
    вернуть результат
конец

вывод(заменить_многое("Привет, мир", [["Привет", "Hello"], ["мир", "world"]]))
```

### Проверка, что строка — число

```
функция это_число(s)
    попытка
        число(s)
        вернуть да
    поймать _
        вернуть нет
    конец
конец

вывод(это_число("42"))      # да
вывод(это_число("abc"))     # нет
```

## Списки и словари

### Уникальные элементы с сохранением порядка

```
конст исходный = [1, 2, 3, 2, 1, 4, 3]
вывод(уникальные(исходный))    # [1, 2, 3, 4]
```

### Сортировка по полю

```
конст люди = [
    {"имя": "Аня", "возраст": 25},
    {"имя": "Боря", "возраст": 22},
    {"имя": "Вася", "возраст": 30},
]

конст по_возрасту = сортировать(люди, (a, b) => a["возраст"] - b["возраст"])

для ч в по_возрасту
    вывод("{ч["имя"]}: {ч["возраст"]}")
конец
```

### Группировка

```
функция сгруппировать(массив, по_ключу)
    перем группы = {}
    для элемент в массив
        конст ключ = по_ключу(элемент)
        если не содержит(ключи(группы), ключ)
            группы[ключ] = []
        конец
        добавить(группы[ключ], элемент)
    конец
    вернуть группы
конец

конст числа = [1, 2, 3, 4, 5, 6]
конст по_чётности = сгруппировать(числа, x => если x % 2 == 0: "чётные" иначе: "нечётные")
вывод(по_чётности)
```

### Поиск максимума по полю

```
функция максимум_по(массив, по_ключу)
    если длина(массив) == 0: вернуть пусто
    перем лучший = массив[0]
    для элемент в массив
        если по_ключу(элемент) > по_ключу(лучший)
            лучший = элемент
        конец
    конец
    вернуть лучший
конец

конст люди = [{"имя": "А", "балл": 5}, {"имя": "Б", "балл": 9}, {"имя": "В", "балл": 7}]
вывод(максимум_по(люди, ч => ч["балл"]))
```

### Объединение словарей

```
функция объединить_словари(a, b)
    перем итог = {}
    для к в ключи(a): итог[к] = a[к]
    для к в ключи(b): итог[к] = b[к]
    вернуть итог
конец
```

## Управление потоком

### Ранний выход из функции

```
функция найти_первое_отрицательное(числа)
    для n в числа
        если n < 0: вернуть n
    конец
    вернуть пусто
конец
```

### Прерывание цикла

```
для i от 1 до 1000
    если i * i > 100: прервать
    вывод(i)
конец
```

### Пропуск итерации

```
для n в [1, 2, 3, 4, 5, 6]
    если n % 2 == 0: продолжить
    вывод(n)    # 1, 3, 5
конец
```

### Цикл с условием выхода в конце

В Ясном нет `do-while`, но можно сделать так:

```
перем готово = нет
пока не готово
    конст ввод_пользователя = ввод("Введите 'выход' для выхода: ")
    если ввод_пользователя == "выход"
        готово = да
    иначе
        вывод("Вы ввели: {ввод_пользователя}")
    конец
конец
```

## Функции

### Функция с переменным числом аргументов

```
    для n в числа
```

```
функция применить_дважды(f, x)

конец
```

### Замыкание (функция, помнящая значения)

```
функция счётчик_от(начало)
```

## Классы и объекты

### Простой класс с методами

```
класс Прямоугольник
    функция создать(ширина, высота)
        это.ширина = ширина
        это.высота = высота
    конец

    функция площадь()
        вернуть это.ширина * это.высота
    конец

    функция периметр()
        вернуть 2 * (это.ширина + это.высота)
    конец
конец

конст p = Прямоугольник(5, 3)
вывод(p.площадь())       # 15
вывод(p.периметр())      # 16
```

### Наследование

```
класс Фигура
    функция создать(цвет)
        это.цвет = цвет
    конец

    функция описание()
        вернуть "Фигура цвета {это.цвет}"
    конец
конец

класс Круг наследует Фигура
    функция создать(цвет, радиус)
        родитель.создать(цвет)
        это.радиус = радиус
    конец

    функция площадь()
        вернуть мат.пи * это.радиус * это.радиус
    конец
конец

конст c = Круг("красный", 5)
вывод(c.описание())     # Фигура цвета красный
вывод(c.площадь())      # 78.5...
```

```
класс Пользователь

перем количество_пользователей = 0
```

## Модули

### Создать модуль

Файл `строки_утилиты.ya`:

```
экспорт функция перевернуть(s)
    перем результат = ""
    для i от длина(s) - 1 до 0 по -1: результат += s[i]
    вернуть результат
конец

экспорт функция повторить_с_разделителем(s, n, разделитель)
    перем итог = ""
    для i от 1 до n
        итог += s
        если i < n: итог += разделитель
    конец
    вернуть итог
конец
```

### Использовать модуль

```

вывод(у.повторить_с_разделителем("ха", 3, "-"))
```

## Часто встречающиеся проблемы

### «Идентификатор не найден» при использовании `пусть`

`пусть` удалён в версии 0.5. Используйте `конст` для неизменяемых
переменных и `перем` для изменяемых.

### «Идентификатор не найден» при использовании `возврат`

Каноническая форма — `вернуть` (глагол повелительного наклонения).

### `//` теперь не комментарий

Только `#`. Если в редакторе поставлен `//`, замените на `#`.

### `json_создать` или `regex_*` не работают

Они теперь модули: `json.создать`, `регвыр.найти`,
`регвыр.заменить`, `регвыр.совпадает`.

### Интерполяция не сработала

Интерполяция `{...}` работает только в двойных кавычках. В одинарных
строки берутся буквально.

```
вывод("Сумма: {2+2}")     # Сумма: 4
вывод('Сумма: {2+2}')     # Сумма: {2+2}
```

### Деление двух целых даёт дробь

Это сделано намеренно: `7 / 2 == 3.5`, не `3`. Для целочисленного
деления используйте `мат.пол(a / b)`.

### Сравнение объектов

В текущей версии равенство объектов и массивов — по содержимому. Для
сложных структур надёжнее сравнивать через `json.создать`.
